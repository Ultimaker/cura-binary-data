Introducing a new engine plugin from the Cura team!

A new plugin has been created by the Cura team that makes use of the improved engine plugin capabilities introduced in Cura 5.5. The plugin is called CuraEngine Tiled Infill Generation and it allows users to select from an expanded list of infill patterns, as well as allowing users to add their own infill pattern, via specification with a *.wtk file.Try it out for yourself
