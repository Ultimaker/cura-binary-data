Export using the new Universal Cura Projects

A new way to export models has been added called Universal Cura Projects (UCP). The option to save a Universal Cura Project can be found in the File menu. Universal Cura Project files can be used on any printer and they allow you to include positional data for the models on your build plate as well as select settings. Visit our Thingiverse page to download this amazing Dinosaur SD Card Holder to try the ease of sharing settings yourself.
