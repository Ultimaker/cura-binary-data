New tree supports by Thomas Rahm.

This special version of UltiMaker Cura contains a new version of tree support that uses significantly less filament, reduces printing time, and is easier to remove than our current tree supports. These changes have been introduced by our amazing community member Thomas Rahm in an effort to improve Cura. We want to help bring his improvements to a bigger audience so it can be enjoyed and tested by more makers like you. This spotlight release highlights this new version of tree supports and contains only changes related to this new version of tree supports.
