Building Ultimaker Cura from source is now more accessible

If you want to join our community as a contributor, it is now easier than ever before. We have made it much easier to build and run Ultimaker Cura from source. Improving the way that Ultimaker Cura is built, and the associated documentation, allows open-source contributors on Mac, Linux, and Windows to help improve Ultimaker Cura. Read more about how to get started here.
